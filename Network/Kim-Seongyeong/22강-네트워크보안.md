### Network Security

- confidentiality
    
    : sender와 receiver 사이에 교환되는 메세지는 둘만 알아야 함(기밀성)
    
- authentication
    
    : 인증
    
- message integrity
    
    : 오차 없이 메세지를 받아야 함
    
- access and availability
    
    : 항시적으로 서비스 제공되어야 함
    

### cryptography (암호화)

- 방식
    - Symmetric key(대칭키)
        
        : sender와 receiver가 동일한 key를 가지고 있어야 함
        
    - Public key
        
        : 두 종류의 키를 사용(public, private)
